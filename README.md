# 📊 College Placement Analysis

## 🎯 Project Overview
Analysis of placement data from 50 engineering and MBA students 
across 5 branches to identify key factors driving placement success.

## ❓ Business Questions Answered
1. Does higher CGPA guarantee placement?
2. Which skills lead to higher salary packages?
3. Do internships increase job offer chances?
4. Which branch has the best placement rate?

## 🛠️ Tools Used
- Microsoft Excel (Pivot Tables, Charts, Dashboard)
- Excel Formulas (AVERAGEIF, AVERAGEIFS, COUNTIF, INDEX+MATCH)

## 📈 Dashboard Preview
![Dashboard](dashboard_screenshot.png)

## 🔑 Key Findings

### Finding 1 — CGPA vs Placement
- Students below 7.0 CGPA have 0% placement rate
- Students above 8.0 CGPA have 90%+ placement rate
- Critical threshold identified at 7.0 CGPA

### Finding 2 — Skills vs Salary
- ML/AI skills command highest salary at 14.5 LPA
- Python is most versatile skill across all branches
- Salary gap between top and bottom skill = 9 LPA

### Finding 3 — Internships vs Job Offers
- 0 internships → only 6% placement rate
- 1 internship → jumps to 92% placement rate
- 2+ internships → 100% placement guaranteed

### Finding 4 — Branch Wise Placement
- CSE leads with 80% placement and 9.2 LPA average
- CIVIL and MECH lowest at 60% placement rate
- Cross-domain skills close the salary gap significantly
